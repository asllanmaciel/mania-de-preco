# Home inicial do produto

**tipo:** feature  
**impacto:** alto  
**módulo:** experiência inicial / frontend institucional

## Resumo executivo

Foi criada a primeira experiência visual do Mania de Preço no navegador, substituindo a página padrão do Laravel por uma home que já apresenta a proposta do produto como SaaS para lojistas com inteligência de preço e controle financeiro.

## Entregas realizadas

- substituição da página inicial padrão por uma landing page com identidade própria
- construção de hero principal com posicionamento de produto e proposta de valor
- organização visual dos módulos centrais da plataforma
- inclusão de seções com roadmap, API e atalhos úteis para testes
- adaptação da experiência para desktop e mobile sem depender de build frontend

## Estratégia aplicada

A entrega foi pensada para transformar a primeira impressão do sistema. Em vez de mostrar apenas que a aplicação está online, a home agora ajuda a comunicar o produto, orientar a navegação inicial e apoiar demonstrações, validações e testes enquanto o painel interno ainda está em evolução.

## Resultado

O projeto já pode ser apresentado no navegador com uma camada visual mais coerente com a proposta de negócio. Isso melhora a percepção de valor, facilita conversas sobre produto e cria uma base concreta para a próxima fase de construção do frontend operacional.
